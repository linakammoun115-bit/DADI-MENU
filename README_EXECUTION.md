# DADI Golden Menu — prêt à exécuter

## Lancer le projet

```bash
npm install
npm run dev
```

Puis ouvre l'adresse affichée par Vite, généralement :

```text
http://localhost:5173
```

## Modifications ajoutées

- Séparation claire : Omelettes / Crêpes salées / Paninis.
- Séparation claire : Crêpes / Gaufres / Pancakes.
- Image locale pour chaque produit dans `public/products`.
- Descriptions automatiques pour les produits sans description.
- Conservation du thème DADI noir/doré.
